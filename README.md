# Project Work – Tecnologia Web per la Sostenibilità d’Impresa (PW 17)

Questo repository contiene il progetto sviluppato per il Project Work del corso
**Informatica per le Aziende Digitali (L-31)**.

## Descrizione
Il progetto consiste nello sviluppo di una **pagina web statica**, realizzata con **HTML5 e CSS3**,
finalizzata alla divulgazione delle attività di sostenibilità e al **download dei report ufficiali**
di un’impresa operante nel settore primario.

## Tecnologie utilizzate
- HTML5
- CSS3

## Funzionalità principali
- Struttura semantica HTML
- Layout responsive basato su CSS Grid
- Organizzazione dei contenuti in card
- Sezione dedicata al download dei report di sostenibilità
- Accorgimenti di accessibilità di base (skip-link e focus visibile)

## Contenuto del repository
- `index.html`: struttura della pagina
- `styles.css`: foglio di stile
- `screenshots/`: immagini utilizzate nell’elaborato per documentare il progetto
